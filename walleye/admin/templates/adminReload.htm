Administration Menu
Start / Restart Honeywall
       
  This section allows you to start / restart a number of the processes that run on your Honeywall. By selecting Restart Honeywall, you will be restarting the following: "Firewall, Snort, Pcap, p0f, sebekd, argus, hflowd.  
 
  Start Bridge
  Restart Honeywall
  Restart IDS Snort
  Restart Pcap Snort
  Restart Snort-Inline
  Restart Firewall
 
  Start the processes listed above automatically when the Honeywall is rebooted.
