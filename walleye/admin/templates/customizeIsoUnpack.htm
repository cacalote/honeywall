Unpack ISO
 
  The first step to make Preconfigured or Customized isos is to install a development environment on your development system's harddrive. The development environment is created using the unpack-iso.sh script located in the top level directory of the CDROM iso. When invoked, it copies the contents of the CDROM iso to the development system, creates hardlinks to the scripts in the CDROM's dev directory that are needed to rebuild the iso, and generates the hwconfig.cache file. The development environent requires around 320Mb of disk space.

For detailed information about customizing an iso, please refer to the Online User's Manual.
