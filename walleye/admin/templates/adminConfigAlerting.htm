  The service Swatch monitors the firewall logs. Email alerts are generated for any outbound activity. This is an indication that a system has been compromised, or some unauthorized activity is occuring. Be sure you allow TCP 25 outbound in the Honeywall Configuration: Remote Management Section, and your mail server accepts mail from this system.
